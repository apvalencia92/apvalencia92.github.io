¿Como utilizar Sweet Alert en Laravel 5.6?

La libreria Sweet alert de Javascript nos permite generar alertas con estilos visualmente atractivos para los usuarios de nuestro sistema. Por eso hoy te enseñare a insertar esta hermosa libreria en tu proyecto Laravel 5.6

Pasos para usar Sweet Alert en laravel 5.6

  1. En nuestra terminal, dentro de nuestro proyecto escribimos composer require realrashid/sweet-alert

  2. Luego de que se haya instalado abrimos el archivo config/app.php importamos la libreria y generamos un alias: 
            'providers' => [
         RealRashid\SweetAlert\SweetAlertServiceProvider::class,
         ]           
        'aliases' => [
        'Alert' => RealRashid\SweetAlert\Facades\Alert::class,
        ]

  3. En el controlador, en este caso UserController.php importamos el namespace del alias que colocamos en el archivo del paso 2 
            use Alert;
        class UserController extends Controller {
        }

  4. En nuestro layout importamos la libreria de sweet-alert 
     < script src="https://unpkg.com/sweetalert2@7.18.0/dist/sweetalert2.all.js > < /script >

  5. En la vista donde queremos mostrar nuestro mensaje usaremos @include('sweetalert::alert')
